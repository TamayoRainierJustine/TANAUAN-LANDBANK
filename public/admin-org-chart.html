Organization chart (web editor)

I-edit ang mga pahina at URLs sa ibaba. Kung may Firebase, mag-sign in para mag-save sa cloud. Kung wala, gamitin ang Offline save (i-download ang JSON at i-upload sa project).

Offline mode (walang Firebase)

Puwede mo nang baguhin ang chart dito. Pindutin ang Download JSON pagkatapos, palitan ang file na public/org-chart-pages.json sa project, tapos i-deploy. Ang mga bagong PNG ay ilagay sa public/assets/ (parehong file name kung posible).

Opsyonal: mag-set up ng Firebase (copy firebase-config.example.js → firebase-config.js) para mag-save nang direkta sa cloud at mag-upload ng larawan.
