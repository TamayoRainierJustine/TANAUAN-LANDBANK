LANDBANK TANAUAN BRANCH

OPENING OF ACCOUNT

Intro guide and process checklist for account opening at LANDBANK Tanauan Branch.

General

Basic Requirements

Address and 2 valid government-issued IDs.

Client Type

Employed / Self-employed / Student

Each has additional documentary requirements.

Online

LANDBANK Mobile Banking

dobs.landbank.com and mobile app onboarding.

General Requirements

Opening of Account

Bring complete and updated identification and address details before branch visit.

📍

Address

  • • Must be complete and updated
  • • House no./street/purok/zone if available
🪪

Valid IDs

  • • 2 valid government-issued IDs
  • • IDs should show complete/updated details
📄

If address is incomplete

If ID address is not updated or “Barangay only,” bring a Certificate of Residency from Barangay with complete address details.

Client Type

Employed / Self-employed / Student

Additional requirements depend on the applicant category.

💼

Employed

  • • Company ID / Certificate of Employment / Pay slip
  • • TIN ID (TIN verification via orus.bir.gov.ph)
🏪

Self-employed (Original & Photocopy)

  • • DTI
  • • Updated Mayor’s Permit
  • • COR-BIR
🎓

Student

  • • School ID
  • • Latest School Registration

Payroll / Pension

For Payroll/Pension Account Opening

Prepare branch-required onboarding documents for payroll or pension account setup.

📨

Required document

  • • Letter of Introduction
ℹ️

Important note

Branch may require other documents for further verification.
